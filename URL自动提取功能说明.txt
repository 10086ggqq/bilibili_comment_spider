## 🔧 技术实现

### 核心函数

```python
def extract_bvid(input_str):
    """
    从输入字符串中提取BV号
    支持多种格式的B站链接
    """
    # 正则表达式匹配
    patterns = [
        r'/video/(BV[a-zA-Z0-9]+)',  # 标准链接
        r'/(BV[a-zA-Z0-9]+)',         # 短链接
        r'bilibili\.com/video/(BV[a-zA-Z0-9]+)',
        r'b23\.tv/(BV[a-zA-Z0-9]+)',
    ]
    
    for pattern in patterns:
        match = re.search(pattern, input_str)
        if match:
            return match.group(1)
    
    return input_str  # 如果都没匹配，返回原输入
```

### 匹配规则

1. **BV号直接匹配**: `^BV[a-zA-Z0-9]+$`
2. **标准链接**: `/video/(BV[a-zA-Z0-9]+)`
3. **短链接**: `b23.tv/(BV[a-zA-Z0-9]+)`
4. **带参数链接**: 自动忽略`?`后的参数

| 方式 | 操作步骤 | 时间 | 便利性 |

**效率提升**: 6倍

## ⚠️ 注意事项

1. **链接完整性**: 确保复制的链接包含完整的BV号
2. **特殊字符**: 程序会自动处理URL中的特殊字符和参数
3. **错误处理**: 如果提取失败，程序会返回原始输入并尝试作为BV号使用
4. **隐私保护**: 链接中的追踪参数（如spm_id_from）会被自动忽略

### 场景2：批量爬取多个视频
```python
# 从浏览器标签页批量复制URL
urls = get_urls_from_clipboard()  # 假设的函数

for url in urls:
    crawl_comments(url, max_pages=5)
    time.sleep(1800)  # 安全间隔
```

### 场景3：脚本自动化
```python
# 监控剪贴板，自动爬取
import pyperclip

last_url = ""
while True:
    current = pyperclip.paste()
    if current != last_url and "bilibili.com/video/" in current:
        print(f"检测到新链接: {current}")
        crawl_comments(current, max_pages=5)
        last_url = current
    time.sleep(5)
```
