### 场景2：批量爬取多个视频

```python
from bilibili_crawler import crawl_comments
import time

# 从浏览器批量复制的链接列表
video_urls = [
    "https://www.bilibili.com/video/BV1ofR5B6E72/?spm_id_from=...",

    # 安全间隔
    if i < len(video_urls):
        print("等待30分钟后继续...")
        time.sleep(1800)
```

### 场景3：监控剪贴板自动爬取

```python
import pyperclip
import time
from bilibili_crawler import crawl_comments

print("监控剪贴板中... (复制B站链接自动爬取)")
last_url = ""

while True:
    current = pyperclip.paste()
    
    # 检测是否是新的B站链接
    if current != last_url and "bilibili.com/video/" in current:
        print(f"\n检测到新链接: {current}")
        
        try:
            comments = crawl_comments(current, max_pages=5)
            print(f"✓ 爬取完成: {len(comments)} 条评论")
        except Exception as e:
            print(f"✗ 爬取失败: {e}")
        
        last_url = current
    
    time.sleep(2)  # 每2秒检查一次
```

## 🔧 技术实现

### 核心函数：extract_bvid()

```python
def extract_bvid(input_str):
    """
    从输入字符串中提取BV号
    支持多种B站链接格式
    """
    input_str = input_str.strip()
    
    # 如果已经是纯BV号，直接返回
    if re.match(r'^BV[a-zA-Z0-9]+$', input_str):
        return input_str
    
    # 从URL中提取BV号的多种模式
    patterns = [
        r'/video/(BV[a-zA-Z0-9]+)',           # 标准链接
        r'/(BV[a-zA-Z0-9]+)',                  # 通用匹配
        r'bilibili\.com/video/(BV[a-zA-Z0-9]+)',  # 带域名
        r'b23\.tv/(BV[a-zA-Z0-9]+)',           # 短链接
    ]
    
    for pattern in patterns:
        match = re.search(pattern, input_str)
        if match:
            return match.group(1)
    
    # 都没匹配到，返回原输入
    return input_str
```

### 匹配规则

1. **直接BV号**: `^BV[a-zA-Z0-9]+$`
   - 匹配: `BV1ofR5B6E72`
   
2. **标准链接**: `/video/(BV[a-zA-Z0-9]+)`
   - 匹配: `https://www.bilibili.com/video/BV1ofR5B6E72/`
   
3. **短链接**: `b23\.tv/(BV[a-zA-Z0-9]+)`
   - 匹配: `https://b23.tv/BV1ofR5B6E72`
   
4. **通用匹配**: `/(BV[a-zA-Z0-9]+)`
   - 匹配所有包含BV号的情况
